# 交付物模式说明

## 1. 高保真原型

目标：
- 模拟真实产品流程
- 让用户能点、能看、能比较

重点：
- 页面状态
- 流程节奏
- 主操作路径
- 信息密度与层级

不要做成：
- 只会展示的静态海报页

## 2. HTML 幻灯片

目标：
- 像演示文稿，而不是像官网首页

重点：
- 16:9 画布
- 翻页节奏
- 章节页 / 信息页 / 图像页的变化
- 适合讲述的主次关系

不要做成：
- 一页塞满信息的网页截图拼贴

## 3. 动效 / 视频风格页面

目标：
- 用节奏与镜头感表达信息

重点：
- 分段推进
- 局部动效，而不是全屏一起飞
- 播放状态持久化

不要做成：
- 华而不实的全局炫技动画
